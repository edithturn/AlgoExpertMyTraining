# Basic Commands System Design

## Client Server Model


netcat  or nc, is command used to any operation in Linux to TCP, UDP, or UNIX domains sockets.

```console
edith@edithcp:~$ netcat -l 8081
edith@edithcp:~$ nc -l 8081

```
```bash
nc 127.0.0.1 8081
# nc   =>  netcat
# -l   => listen
# 8081 => port

```
